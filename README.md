# Spring H2 Demo

REST API project developed in Java using Spring Boot framework and H2 database.

---

## Requirements

- Java 11
- Spring Boot 2.7.8
- Apache Maven 3.8.6

---

## Run App

Run the SpringH2DemoApplication class as Java Application.
